# 📌 Project 3 - Tabbed Content Section

A modern and responsive **Tabbed Content Section** built using HTML, CSS, and JavaScript. This project allows users to switch between different content sections using tabs.

---

## 🚀 Features

- Multiple tabs (Overview, Info, Contact)
- Dynamic content switching using JavaScript
- Active tab highlighting
- Smooth transition animations
- Responsive design (mobile-friendly)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
